# Projeto Login

Tela de login simples feita com HTML, CSS e JavaScript puro — sem dependências externas.

## Como abrir

1. Extraia a pasta
2. Abra no VS Code: `File > Open Folder`
3. Instale a extensão **Live Server** (se não tiver)
4. Clique com o botão direito em `index.html` → **Open with Live Server**

> Ou simplesmente dê dois cliques em `index.html` no seu explorador de arquivos.

## Funcionalidades

- Validação de e-mail e senha em tempo real
- Mostrar/ocultar senha
- Lembrar e-mail no `localStorage`
- Feedback de erro e sucesso
- Simulação de requisição (delay de 1.2 s)
- Layout responsivo
